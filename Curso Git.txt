Comandos Git bash 
ls -l Se ven muchas cosas
clear borrar comandos
cd { Nombre del disco } para cambiar el disco donde se coge el almacenamiento
pwd ver en qué disco se está trabajando
cd… para volver al directorio anterior
touch <Archivo>.txt para crear archivo texto plano
cat para ver el contenido de un archivo
si pones !y el número de un comando que aparece en history se ejecutará
si pones rm y un archivo este se eliminará
rm -f eliminarlo aunque sea importante
comando - - Help te das opciones que puedes hacer con r
git init se empiezan a ejecutar comandos de git
git status ver la branch  en la que estas los comits y las filas que tienes
git add creas un commits 
git rm –cached cancelar guardado 
git checkout para cambiar de rama o de commit
poner git add . para aceptar los cambios en los commits o crear un comit
git log para ver el historial de comits
